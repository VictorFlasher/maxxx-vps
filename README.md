# Maxxx-Local Chat

Безопасный мессенджер с端到端 шифрованием.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Скопируйте `.env.example` в `.env` и настройте:

```bash
cp .env.example .env
```

**Обязательные параметры:**
- `SECRET_KEY` — случайная строка (минимум 32 символа)
- `DB_PASS` — пароль от PostgreSQL
- `DB_NAME` — имя базы данных (по умолчанию `postgres`)
- `DB_USER` — пользователь БД (по умолчанию `postgres`)

### 3. Инициализация базы данных

**Важно!** Перед первым запуском необходимо создать схему и таблицы:

```bash
psql -U postgres -d postgres -f app/init_db.sql
```

Или через pgAdmin/другой клиент выполните содержимое `app/init_db.sql`.

### 4. Запуск приложения

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Откройте в браузере: http://localhost:8000

## 🔒 Безопасность

- ✅ JWT аутентификация с проверкой exp
- ✅ Защита от XSS (textContent вместо innerHTML)
- ✅ CSP заголовки
- ✅ Rate-limiting на WebSocket (5 подключений/пользователь)
- ✅ Пул соединений к БД
- ✅ Хэширование паролей (bcrypt)
- ✅ Параметризованные SQL-запросы

## 🛠️ Разработка

### Логирование

Логи сохраняются в `logs/app.log` с ротацией (макс. 10 МБ, 5 файлов).

### Health Check

Эндпоинт для проверки работоспособности:
```
GET /health
```

Ответ: `{"status": "healthy"}`

## ⚠️ Важно

- **Не коммитьте `.env` в Git!** Файл добавлен в `.gitignore`
- Для production используйте HTTPS (см. инструкцию в `app/init_db.sql`)
- При масштабировании включите Redis (`REDIS_ENABLED=true`)
